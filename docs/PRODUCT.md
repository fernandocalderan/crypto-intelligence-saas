# Product

## Propuesta

Crypto Intelligence es un SaaS para convertir datos de mercado crypto en señales accionables con contexto, scoring y explicación clara para traders y desks.

## MVP

- dashboard con activos seguidos y señales activas
- login base listo para integrar autenticación
- pricing para producto self-serve o desk
- endpoints mock para validar UX antes de conectar datos reales

## Usuario objetivo

- traders activos que quieren filtrado de ruido
- pequeños desks que necesitan señales explicables
- operadores que valoran rapidez antes que suite institucional compleja

## Qué no entra todavía

- ejecución automática
- portafolios multiusuario complejos
- conectores on-chain y derivados en tiempo real
- billing completamente conectado
